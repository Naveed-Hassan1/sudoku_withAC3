# sudoku_withAC3
This Python-based Sudoku solver leverages Constraint Satisfaction Problem (CSP) techniques—specifically AC-3, Forward Checking, and MRV-guided Backtracking—to highly optimize the search process and efficiently solve puzzles of all difficulty levels.

This Python program is an optimized Sudoku solver that treats the puzzle as a Constraint Satisfaction Problem (CSP) to efficiently solve boards of varying difficulties. It integrates three major AI techniques: AC-3, which logically pre-checks the board to enforce arc consistency and prune impossible values; Backtracking Search utilizing the Minimum Remaining Values (MRV) heuristic to intelligently guess the most constrained cells when pure logic stalls; and Forward Checking, which instantly updates neighboring cells to prevent invalid future assignments. By maintaining arc consistency during the search, these modules work together to drastically reduce the search space, allowing the program to find solutions with minimal computational overhead and backtracking failures.
